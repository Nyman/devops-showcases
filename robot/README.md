## Installation

* [Official instructions in robotframework.org](http://code.google.com/p/robotframework/wiki/Installation)

### OSX

* Install [Homebrew](http://brew.sh/)

* Run in Terminal

    brew install python
    pip install robotframework
    pip install robotframework-selenium2library
